**VoteSecure Lite** is a robust, secure online voting system simulation built with **Python** and **Flask**. Designed for educational and demonstration purposes, it showcases how to implement complex security features‚Äîsuch as **OTP Multi-Factor Authentication** and **strict password validation**‚Äîwithout incurring external costs.

## üöÄ Project Overview

The core mission of VoteSecure Lite is to demonstrate a fair digital ballot handling system. It ensures that every transaction is atomic and verifiable. The latest version introduces advanced authentication workflows that mimic enterprise-grade systems:

-   **Mock Email Inbox (Zero-Cost Testing)**: To simulate real-world email OTPs without paid SMTP servers or complex API keys, the system includes a built-in "Virtual Inbox". This allows developers and testers to see the OTP emails generated by the system instantly within the application itself.
-   **Multi-Factor Authentication (OTP)**: Users must verify their email address via a One-Time Password (OTP) before their account is activated.
-   **Enhanced Password Security**: The system enforces strict password complexity rules (uppercase, lowercase, numbers, and special characters) using Regex validation to prevent weak credentials.
-   **Secure Voting Transaction**: Ensures adherence to the "One Person, One Vote" principle using database locking mechanisms and atomic commits.
